# AutoCAD-BlockBox Folder Structure

Recommended Dropbox structure:

```text
Dropbox
└── Cannon Construction
    └── AutoCAD
        └── 5 - BLOCKS
            ├── BLOCKS-Dev
            │   ├── Dynamic
            │   ├── Static
            │   ├── package-definitions
            │   ├── changelogs
            │   ├── docs
            │   └── tools
            │
            ├── BLOCKS-Release
            │   ├── Dynamic
            │   ├── Static
            │   └── Catalogs
            │
            ├── Releases
            │   ├── Current
            │   └── Archive
            │
            ├── Documentation
            └── Templates
```

## Rule

BLOCKS-Dev is the working area.

BLOCKS-Release is production output only.
